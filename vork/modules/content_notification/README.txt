Email Content Notification
------------

Description
-----------

This is the advance version of Admin Content Notification
(https://www.drupal.org/project/admin_content_notification) using this module
user can create as many email notification reactions as needed in the website on
content creation/updation.

Install
-------

Copy and paste downloaded module into modules folder.
Enable the module.
Open "/admin/content/content-notification" and configure the settings.
